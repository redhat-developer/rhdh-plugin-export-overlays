# RHIDP-14515: Rename homepage frontend plugin from dynamic-home-page to homepage

- **Status:** Review
- **Assignee:** Christoph Jerolimov
- **Team:** RHDH Frontend Plugins & UI

## Background

The homepage frontend plugin is currently named `dynamic-home-page` â€” a legacy name from when it was a thin wrapper around the upstream Backstage `home` plugin with added dynamic mount points. With RHDH 2.x / NFS, the scalprum dependencies and mount point code are no longer needed. The backend counterpart is already named `homepage-backend`, so the frontend should be aligned to just `homepage`.

The plugin ID also changes from `home` to `homepage` so it can be installed alongside the upstream home plugin.

All upstream prerequisites in rhdh-plugins are already complete (package renamed, plugin ID changed, scalprum removed, OFS tests removed, package published).

## Scope

This plan covers **only the overlay repo** (`rhdh-plugin-export-overlays`). Scorecard E2E test updates are out of scope and will be handled in a follow-up PR.
